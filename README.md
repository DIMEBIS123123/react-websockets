<h1>💫Paint на REACT + WEBSOCKETS</h1>
<h6>♥️Демо-ролик проекта можно посмотреть в самом низу!♥️</h6>
<p>
	Сделал Онлайн Рисовалку Paint где пользователи коннектятся по id сессии и в рамках одной сессии могут рисовать одновременно, конечно же стилизованно под Made In Abyss в стиле Бездны сверху вниз
</p>
<h2>🔦 Технологии с которыми работал</h2>
<ul>
	<li>React.js</li>
	<li>Mobx</li>
    <li>Bootstrap</li>
	<li>Scss</li>
	<li>Node.js</li>
	<li>Express</li>
	<li>WebSockets</li>
    <li>REST API AXIOS</li>

</ul>
<h2>🦊 Основные функции что выполнимы в проекте </h2>
<ul>
    <li>Генерация уникальной сессии при запуске приложения</li>
    <li>Отслеживание пользователей в рамках одной сессии</li>
    <li>Генерация уникального имя пользователя если он не ввёл его</li>
    <li>Набор разных инструментов для рисования(Кисточка Круг Прямоугольник Линии Ластик)</li>
    <li>Настройки толщины, обводки, заливки</li>
    <li>Очистка синхронизированная</li>
    <li>Undo и Redo функционал, где пользователь может отменять и возвращать именно своё действие, не влияя на других пользователей</li>
    <li>Сохранение рисунка пользователя по кнопке</li>
    <li>И всё это одновременно могут делать пользователи в рамках одной сессии</li>
</ul>
<h2>🔆 Что я углубил в рамках разработки</h2>
<ul>
    <li>Я сильно углубился в понимание canvas и того как реализуется paint с нуля</li>
    <li>Работу с состояниями и Mobx, подход где вместо глобального контекста используется работа с самим классом по импорту</li>
    <li>Middleware, работа с лимитом</li>
    <li>Изучил что такое WebSockets впринципе и то как с этим работать, отслеживать сообщения на сервер и обрабатывать по случаю(в рамках Paint у нас много событий и это был крутой опыт)</li>
    <li>Больше понимания по JSON, как легко происходит отправка данных, картинок, обьектов и тд преобразуя и преобразуя обратно)</li>
</ul>

<h2>🪝 Как локально запустить этот проект у себя</h2>
<ul>
    <li>1. Клонировать репозиторий</li>
    <li>2. Установить зависимости в серверную папку и клиентскую через: npm install</li>
	<li>3. В Терминале открыть папку сервера и написать npm run dev, затем так-же открыть папку клиента и написать npm start</li>

</ul>
